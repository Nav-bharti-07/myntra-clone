# Myntra Clone
A fully responsive Myntra home page clone built using HTML and CSS.

## Features
- Responsive navigation bar
- Search bar
- Profile, Wishlist, and Bag icons using Material Icons
- Banner and Category section
- Clean footer design

## Technologies Used
- HTML5
- CSS3
- Font Awesome & Material Symbols
